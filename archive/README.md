# Archive

This folder holds historical or superseded Company Agent material that has been absorbed into active architecture or intentionally retired.

## Folders
- `superseded/` contains documents that were replaced by active `AGENTS.md`, `memory/`, `skills/`, or `skills/*/reference/` material.

## Superseded Documents
- `superseded/agent_protocol.md` was absorbed during Phase 2 Batch 1.
- `superseded/company_operations_overview.md` was absorbed during Phase 2 Batch 2.
- `superseded/ai_operating_interface_guide.md` was absorbed during Phase 2 Batch 3.

## Rules
- Do not use archive files as active operating instructions.
- Do not move additional files here unless a migration batch explicitly scopes that archival work.
- If archived material is consulted, treat it as historical context and prefer active source-of-truth files.
