- Supabase/Postgres remains canonical structured truth.
- Memgraph is the derived relationship graph, not a canonical replacement.
- Retrieval should add cited source context through pgvector or the current retrieval baseline.
- Markdown/GitHub repos should hold governance, durable meaning, decisions, operating rules, and short readbacks.
- LangGraph-style orchestration should route multi-step agent questions and review pauses.
- App UI/helpers should become the operator control surface when implementation is approved.
- Agents propose; humans/domain owners approve canonical writeback.
